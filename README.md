# 杀戮尖塔2 Mod —— 极速战斗

## 功能

- 加速战斗场景
- 跳过卡牌动画
- 移除等待限制

## 安装

1. 下载本Mod
2. 解压到游戏根目录  
（确保 `Slay the Spire 2\mods\InstantSpeed` 目录下存在 `InstantSpeed.dll` 及 `InstantSpeed.pck`）
3. 进入游戏，自动生效，无需任何设置  
（第一次使用Mod需要手动确认，主界面右下角可以看到Mod加载数量）

## 卸载

删除 `Slay the Spire 2\mods\InstantSpeed` 目录及子文件即可

## Features

- Accelerated Combat
- Animation Skipping
- Zero Wait Times

## Installation

1. Download the mod files.
2. Extract the contents into your game’s root directory.  
(Ensure that the folder `Slay the Spire 2\mods\InstantSpeed` contains both `InstantSpeed.dll` and `InstantSpeed.pck`.)  
3. Launch the game. The mod will activate automatically without requiring further configuration.  
(For first-time users, a manual confirmation prompt may appear. You can verify the mod is active by checking the "Mods Loaded" count in the bottom-right corner of the main menu.)

## Uninstallation

To remove the mod, simply delete the InstantSpeed folder and its subfiles located at:
`Slay the Spire 2\mods\InstantSpeed`
